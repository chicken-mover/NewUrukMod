# NewUrukMod
New Uruk themed EUIV mod

## Installation

Checkout to:
```
MyDocuments/ParadOx Interactive/Europa Universalis IV/mod/
```

Add text file `NewUrukMod.mod`:
```
name="New Uruk"
path="mod/NewUrukMod"
supported_version="1.12"
replace_path = "history/provinces"
replace_path = "common/bookmarks"
```
